元组和集合

元组

元组存储数据的方式和列表很像，但是元组内的元素是不能变更的。

基本操作

创建元组

tp = (1,2,3,4,5)
tp1 = (1, 'a', [3,4,5], True)
"""
只有一个元素时，需要在第一个元素后加上一个`,`
如果不加`,`，系统会认为创建的不是元组，这个元素是什么类型，就会被认为创建的是什么类型
"""
tp2 = ('A',) 
tp3 = ([1,2,3],)

索引和切片

索引和切片的操作与列表和字符串一致。

探讨

元组真的是不可变的吗？

tp = (1, 2, 3)
tp1 = (1, 'a', [3,4,5], True)

集合

集合(set)是一个无序的不重复元素序列。 集合中的元素不会重复，并且可以进行交集、并集、差集等常见的集合操作。 可以使用大括号 { } 创建集合，元素之间用逗号 , 分隔， 或者也可以使用 set() 函数创建集合。

创建集合

s = {1,2,3,4,5}

添加元素

add(x)函数，x必须不可变

>>> s.add(6)
>>> s.add((1,2,3))

update(x)函数，x可以是多个序列，但是必须是序列。

>>> s.update('abafdjlk')

移除元素

  • remove(x)移除元素x，如果x不在集合里，则报错
  • discard(x)移除元素x，如果x不在集合里，不会报错
  • pop() 随机删除任意一个元素

方法

  1. difference() 取差集
>>> s= {1,2,3,4}
>>> s1 = {4,5,6,7}
>>> s.difference(s1)
{1, 2, 3}
  1. intersection() 取交集
>>> s= {1,2,3,4}
>>> s1 = {4,5,6,7}
>>> s.intersection(s1)
{4}
  1. union() 取并集
>>> s= {1,2,3,4}
>>> s1 = {4,5,6,7}
>>> s.union(s1)
{1, 2, 3, 4, 5, 6, 7}
